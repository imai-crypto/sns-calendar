# SNS投稿カレンダー

Excelファイルをアップロードして、投稿スケジュールをカレンダー形式で可視化するツールです。

## 使い方

1. [カレンダーを開く](https://あなたのユーザー名.github.io/sns-calendar/)
2. 「Excelアップロード」ボタンをクリック
3. Excelファイル（.xlsx/.xlsm）を選択
4. カレンダーに投稿データが表示されます

## 対応フォーマット

Excelファイルの「全体カレンダー」シートに以下の列が必要です：

| 列名 | 説明 |
|------|------|
| 日程 | 投稿日（必須） |
| 投稿プラットフォーム | IGリール、IGストーリー、IGフィード、TikTok |
| 作品名 | 作品/プロジェクト名 |
| 投稿名 | 投稿タイトル |
| ブランド | ブランド名 |
| 時間 | 投稿時間 |

## 機能

- 📅 月次カレンダービュー
- 🔍 媒体フィルター（IGリール/IGストーリー/IGフィード/TikTok）
- 🎬 作品名フィルター
- 📱 レスポンシブデザイン
